# rpg-sage-creative/template-literal-utils

This repo is designed to assist in test logging.

The `tagLiterals` command is used to make logging code simpler while also making logging output more readable.

# v1.2.0

As of v1.2.0 this repo assumes you are using pnpm (instead of npm) and that you are making use of the updated `v1.3.0` version of `dev-scripts`.
